# 模板

  • 主要针对 C 泛型编程和 STL 技术，模板就是建立通用的工具，大大提高复用性，但不能直接使用，它只是一个框架，C 提供两种模板机制：函数模板和类模板 。

# 函数模板

  • 作用： 建立一个通用的函数，函数的返回值类型和形参类型可以不具体制定，用一个虚拟的类型代表。
    语法：template <typename T>
    说明：template 表示声明创建模板， typename 表示其后面的符号是一种数据类型，可以用 class 代替 ，T 表示通用的数据类型，名称可以随意替换，通常为大写字母 ，函数模板有两种方式 ：自动推导类型和显示指定类型

注意：自动推导类型，必须推导出一致的数据类型 T 才可以用，模板必须要确定出 T 的数据类型，才可以使用

  • 函数模板调用时，如果利用自动类型推导，不会发生隐式类型转换
