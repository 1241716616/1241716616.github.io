# 无重复字符的最长子串

# 解法一

  • 解题思路：无重复字符，所以直观思路是需要一个 hash 表存储子串，然后添加新字符的时候判断 hash 表中是否含有该字符，所以我们可以计算从每个元素开始的无重复子串的长度，然后取出其中最长的即可，最坏的时间复杂度为 O (n^2)。

# 解法二

  • 但是这种做法时间消耗太大，有没有什么办法能够一次遍历即可，因为两次遍历其实遍历了很多重复的元素，所以我们可以使用左右指针分别指向子串的首尾，如果加入新元素不重复，那么右指针就一直右移，如果加入新元素重复了呢？那么右指针就先歇歇，左指针右移，然后移除前面的元素。

举个例子：

p w w k e w

  1. 刚开始左右指针都指向 p, 右指针最远移到 w，此时长度为 2；
  2. 然后左指针右移到 w，右指针又可以移动到 e, 此时长度位 3；
  3. 然后左指针右移到 k，右指针移动到 w 结束。

在这个过程中，左右指针都最多遍历整个字符串 1 遍，所以时间复杂度为 O (n)。

# 解法三

  • 解题思路：开辟了一个长度为 256 的向量，默认值为 - 1 。以查表的形式统计出现过的字符，出现过的字符会获得 i，i - start 为子串长度。

# 删除字符串中的所有相邻重复项

  • 解题思路：栈模拟；对于这种的删除隐式重复字母的问题，什么叫隐式，即 abbaabba 叫隐式，把 bbbb 抹掉后，aaaa 也成为了含重复字母的字符串，故也要删掉。抛开栈思想不说，我们简单模拟一下字母序列的变化过程 abb -> a__ -> a__a -> aa -> ""，这就像在玩小游戏消消乐，遇到相同颜色的就一起消掉，只不过方向单一化罢了。把 aa 放到容器中，然后来了个 b，又来了一个 bb，之后消掉这两个 bb，最后的 aa 来了，把之前的 aa 也消掉。那用什么数据结构来模拟这个装字母的容器呢？数组、栈都行。

# 删除字符串中的所有相邻重复项 II


  • 解题思路：辅助栈；跟踪栈顶元素的出现次数。按照问题要求操作：对重复的相邻字母计数，当计数达到 k 时将其删除，重复此操作，直到没有删除的字符为止。
