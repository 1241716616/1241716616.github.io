# 一些小的常用综合案例练习

# 滑动条的创建和使用

滑动条是 OpenCV 动态调节参数特别好用的工具，依附窗口而存在。OpenCV 没有实现按钮的功能，我们可以用仅含 0 -1 的滑动来实现按钮的按下、弹起效果。

函数原型： int createTrackbar(conststring& trackbarname, conststring& winname,int* value ,int count , TrackbarCallback onChange = 0,void* userdata = 0 );

  • 参数 1：轨迹条的名字
  • 参数 2：窗口的名字，表示窗口依附到哪个窗口上。
  • 参数 3：一个指向整型的指针，表示滑块的位置，在创建时，滑块的初始位置就是该变量当前的值。
  • 参数 4：表示滑块可以达到的最大位置的值，滑块的最小位置的值始终为 0 。
  • 参数 5：TrackbarCallback 类型的 onChange ，默认值为 0 ，是一个指向回调函数的指针，每次滑块位置改变时，这个函数都会进行回调。并且这个函数的原型必须为 void XXX (int , void*) , 其中第一个参数是轨迹条的位置，第二个参数是用户数据（看下面的第六个参数）。如果回调是 NULL 指针，表示没有回调函数的调用，仅第三个参数 value 有变化。
  • 参数 6：是用户传给回调函数的数据，用来处理轨迹条事件。如果使用的第三个参数 value 实参是全局变量的话，可以不去管这个 userdata 参数。

# 鼠标操作

和滑动条的消息映射方式很类似，都是通过一个中介函数配合一个回调函数来实现的，鼠标操作消息回调函数为 SetMouseCallback 。

函数原型： void setMouseCallback (conststring& winname, MouseCallback onMouse ,void* userdata = 0 );

  • 参数 1：窗口的名字
  • 参数 2：MouseCallback 类型的 onMouse ，指定窗口每次鼠标事件发生的时候，被调用的函数指针。这个函数的原型的大概形式为 void Foo (int event, int x, int y, int flags , void* param) 。 event 是 EVENT_+ 变量之一， x 和 y 是鼠标指针在图像坐标系（注意：不是窗口坐标系）中的坐标值， flags 是 EVENT_FLAG 的组合， param 是用户定义的传递到 SetMouseCallback 函数调用的参数。例如：EVENT_MOUSEMOVE 为鼠标移动消息 、EVENT_LBUTTONDOWN 为鼠标左键按下消息等。
  • 参数 3：用户自定义的传递到回调函数的参数。默认值为 0。
