# 对象计数

农业领域经常需要计算对象额数或者在其它领域拍照自动计算，可提高效率，降低成本。现在需要统计计算图片中玉米的颗粒数。

方法：二值分割 + 形态学 + 距离变换 + 连通区域计算


效果图：

  • 类似问题的解决思路：首先寻找一个合适的二值化方法，二值化之后看要不要做形态学操作（如果通过形态学操作后效果很好，能把对象分离出来，即没有相连，独立出来），再做距离变换，邮件里距离变换再选择合适的二值化方法再做二值化，再对它进行分割，最终分割出来的二值图像，因为要统计它的个数，通过 findContours () 寻找轮廓，连通区域，统计连接矩形就可以实现。
