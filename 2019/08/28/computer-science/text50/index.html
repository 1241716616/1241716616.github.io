# BeautifulSoupa 库介绍

  • 和 lxml 一样，Beautiful Soupa 也是一个 HTML/XML 的解析器，功能是用来实现如何解析 HTML/XML 数据。lxml 只会局部遍历，而 Beautiful Soupa 是基于 HTML DOM 的，会载入整个 DOM 树，所以时间和内存开销会很多，性能要低于 lxml。BeautifulSoupa 解析 HTML 比较简单，支持 CSS 选择器、python 标准库中的 HTML 解析器，也支持 lxnl 的 XML 解析器。
