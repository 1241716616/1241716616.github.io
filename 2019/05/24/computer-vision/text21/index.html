# 用 Canny 算子检测图像轮廓

介绍： 使用 Canny 算子，需要指定两个阈值，基于两个阈值获得二值分布图的策略（滞后阈值化）。在低阈值边缘分布图上值保留具有连续路径的边缘点，同时把那些边缘点连接到属于高阈值边缘分布图的边缘上，高阈值的边缘点被保留下来，低阈值分布图上的孤立点全部被移除。
