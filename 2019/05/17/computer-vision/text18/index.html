# 图像滤波

当我们看一幅图像时，是在观察图像中不同灰度级别（或彩色）组成的图案。图像之间的区别，在于它们有不同的灰度分布方式。观察它们的变化频率的这种特征称为频域，频域分析把图像分解成从低频到高频的频率成分，图像强度变化慢的的区域只包含低频率，强度值变化快的区域产生高频率。

# 低通滤波器（线性）

  • 目的是减少图像变化的幅度，方法是把每个像素的值替换成它周围像素的平均值，如果需要让领域内较近的像素具有更高的重要度，可以计算加权平均值，即较近的像素比较远的像素具有更大的权重。

  • 算法原理：如果一种滤波器是用领域像素的加权累加值来替换像素值，这种滤波器就是线性的，如均值滤波器，即将矩形领域内的全部像素累加，除以该领域的数量（即求平均值）。应用一个线性滤波器相当于将内核移动到图像的每个像素，并将每个对应像素乘以它的权重，这个运算在数学上称为卷积。低通滤波器的最终效果是使图像更加模糊或更加平滑，因为它减弱了高频成分，而高频成分正好对应了物体边缘处的快速视觉变化。

# 用低通滤波器进行缩减像素采样

  • 实现原理：图像处理过程中经常会有需要调整图像精度（重新采样）的情况，降低图像精度的过程为缩减像素采样，提升图像精度的过程为提升像素采样，这些算法的难点在于尽可能地保持图像质量，通常采用低通滤波器来实现。

  • 实现：要缩小一幅图像，简单的消除图像中的一部分行和列得到的图像效果很差，当图像中包含高频成分时，由于图像太小而无法包含时，就会出现伪影，也就是空间假频现象。因为在小图像（像素较少的图像）中展现精致纹理和尖锐边缘的效果不如在较高分辨率的图像中展现它们的效果好。图像中精致的细节对应着高频，因此需要在缩小图像之前去除它们的高频成分。

# 中值滤波器（非线性）

  • 实现原理：是非线性的，所以不能用核心矩阵表示，也不能进行卷积运算。它是把当前像素和它领域组成一个集合，然后计算出这个集合的中间值，作为当前像素的值。中值滤波器对消除椒盐噪声非常有用，还有利于保留边缘的尖锐度。

# Sobel 算子

  • 是离散微分算子，用来计算图像灰度的近似梯度。
  • 功能集合了高斯平滑和微分求导
  • 是一阶微分算子，求导算子在水平和垂直两个方向上求导，得到图像 x 方向和 y 方向梯度图像。
  • Sobel 算子的内核中既有正数又有负数，计算结果通常是 16 位的有符号整数图像（CV_16S）, 为了把结果显示为 8 位图像，用数值 0 代表灰度 128 负数表示更暗的像素，正数表示更亮的像素。
