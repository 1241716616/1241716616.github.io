# PyTorch 开发环境的搭建

# 开发环境

  1. Python3.7 + Anaconda 5.3.0
  2. CUDA 10.2
  3. Pycharm Community

# Anaconda 安装

在官网下载后傻瓜式安装即可，注意勾选 Add Anaconda to my PATH enviroment variable ，忘记了可以自己在系统 - 高级系统设置自己添加你安装的路径到环境变量中。安装了 Anaconda 会自带了 Python3.7 .


# Anaconda 测试

cmd 进入 Windows 命令处理程序，执行 conda list , 可以看到已经安装的各种包

执行 conda -V 可以查看 conda 的版本。

# CUDA 安装

需要注意的是 CUDA 只能运行在安装 NV 的显卡上，所以要确定你的电脑有一张显卡，在官网上下载 Windows - x86_64 - 10 - exe [local] ，大概二点几个 G，下载完成后双击运行即可，安装的过程它会进行一系列的系统检查，安装的过程中屏幕会有偶尔的闪动。

# CUDA 测试

安装会默认安装到 C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.2\bin 路径下，下面有一个 nvcc.exe 程序，它就是 CUDA 语言的编译器，同样的运行 cmd 进入 Windows 命令处理程序 ，执行 nvcc -V 如下图 。

可能会出现 nvcc 找不到的情况 ，是因为环境变量没配好，自己手动把 C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v10.2\bin 路径添加进去即可。

# PyTorch 安装

打开 PyTorch 官网 ，依次选择如下图的选项。

复制 conda install pytorch torchvision cudatoolkit=10.1 -c pytorch 命令 ，运行 cmd 进入 Windows 命令处理程序 （注意要以管理员的身份打开），把复制的命令粘贴进去，大概十多秒之后会提示你要不要安装，选择是 （y），就可以了。

# Pycharm 安装

在官网选择社区版下载，下载完后双击运行即可，安装完后，开始会有用户界面习惯的选择，大家可以自己选择喜欢的风格。

# 配置 Interpreter

打开 Pycharm ，新建一个工程后，选择 File - Settings 。配置 python 的编译器，因为 Pycharm 只是一个开发工具，具体编译器还需要我们安装的 Anaconda 安装的 python 。注意 python3.7.exe 是你安装 Anaconda 路径下的。


# 测试

在项目中新建一个 .py 文件。鼠标右击工程项目 - New - Python File ，随意命名，确认即可。
