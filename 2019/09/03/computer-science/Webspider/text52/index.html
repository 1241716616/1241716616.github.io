# lxml 包没有 etree 模块的解决办法

# 各种包的安装步骤

说明： 安装方法在前面的网络请求文章中有写过。在 PyCharm 直接安装就可以了。在 file-->setting-->Project Interpreter，找到 python 解释器安装界面。要安装其它库也可以用这个方法。


说明： 如果安装了 lxml 模块后，from lxml import etree 这样子引入 etree 模块还提示找不到指定模块，是因为 python 3.5 之后的 lxml 中不再有 etree。

解决办法：
方法 1：
虽然新版本无法直接 from lxml import etree 这样，但是它只不过是换了一个办法引出 etree 模块而已，可以使用下面方法引出试一下。


方法 2：
（1）检查你的 python 版本，我的 pycharm 中配置了 python3.7 的解释器，你也可以在文章的末尾处下载 python3.5 进行安装。
（2）配置好了 python3.5 以后，先把 pip 升级到最新版本，下载早期版本的 lxml 包，找到对应的 Python version 3.5 版本下载安装即可，文章末尾处有提供安装包。
